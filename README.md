## Project setup

```bash
$ git clone https://github.com/RetrooFan/code-and-pepper.git

$ npm i

$ cp .env.example .env
```

## Compile and run the project

```bash
# development
$ ./RUNME-dev.sh

# production mode
$ npm run start:prod
```

## Run tests

```bash
# e2e tests
$ ./RUNME-test.sh
```
